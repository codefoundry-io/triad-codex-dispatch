# R11 final functional repair amendment

## Why R10 was invalid

R10 had three valid legs and one schema-invalid Google Scope B leg. The valid Codex Scope A leg
found one normal-use Major: default bootstrap-generated provider launchers did not force audit
prompt redaction, although the README describes persistent audit records as redacted.

The invalid Google result emitted one finding object instead of the required complete
`FormalReview` envelope. Its advisory claim about native Codex field names was false, but two
consecutive Scope B schema failures demonstrated a real output-compliance weakness in the AGY
prompt assembly order.

## Minimal repairs

1. Every generated provider launcher sets `TRIAD_AUDIT_REDACT_PROMPTS=1` in its constructed
   environment. Launcher ownership recognition accepts the new exact grammar while retaining
   shipped legacy upgrade recognition.
2. For a sealed AGY call, trusted packet identity is appended to the user request before the
   existing schema wrapper is applied. The canonical JSON-only schema instruction and final
   `JSON:` marker therefore remain last.

These changes do not enable containment, no-prompt mode, or additional provider retries. A
schema-invalid formal call remains terminal.

## Scope controls

- Ignore already deferred rare same-UID filesystem races and injected low-level faults.
- Keep product skill files unchanged.
- Because wrapper prompt assembly changed, perform one focused `skill-prompt-review` pass on that
  composition only.
- Refresh the real AGY and fresh-Codex token spikes and bind the final executable hashes into the
  R11 candidate.

## Required gate

After complete macOS and pinned Ubuntu 24.04 verification, freeze a new packet and admit all four
legs before consuming results:

- two independent exact `Gemini 3.1 Pro (High)` AGY legs, sequentially queued;
- two independent fresh `gpt-5.6-sol` / `high` Codex legs with no inherited history.

Only a reproducible normal-use Critical or Major defect, genuine evidence-backed conflict, or a
broken required route blocks R11. Reconcile evidence rather than votes.
